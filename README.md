# Rainmeter
使用桌面美化软件“Rainmeter“和”桌面日历“两个工具，打造“高级”电脑桌面，如下图：

![效果图](https://github.com/Jupiter-19/Rainmeter/blob/main/figure/%E5%B1%95%E5%BC%80.png)

## 功能

- 左上角：实时时间
- 左上角：常用文件夹
- 左上角：Chrome和常用网站
- 左上角：常用软件
- 左下角：实时天气和天气预报
- 上方：较常用软件，点击中间可收起
- 右上角：系统监控，CPU占用和WIFI强度
- 右侧：备忘录
- 右侧：系统监控，硬盘、CPU、内存、磁盘、网络传输
- 右下：桌面日历
- 右下：音乐
